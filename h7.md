# c) Automaattinen analyysi. Mitä tässä näytteessä tapahtuu? Mitä tunnisteita (id yms) löydät? Converted_433.92M_2000k.cs8. Analysoi näyte 'rtl_433' ohjelmalla.

Näytteen sai auki komennolla rtl_433 Converted_433.92M_2000k.cs8

<img width="749" height="95" alt="image" src="https://github.com/user-attachments/assets/6d7a2088-41ec-4178-a816-7996b0e6338c" />

Näytteesä on kaapattu ilmeisemmin joku kaukosäädin

* ID / House Code: 8785315 – (24 bittinen koodi) tämä on laitteen/järjestelmän yksilöllinen tunniste.
* Unit: 0 tai 3 – yksikkönumero ( kertoo komennon kohteen, esim unit 0 = olohuoneen lamppu, unit 1 = makuhuoneen lamppu jne)
* Channel: 3 – kanava
* Command/State: ON / OFF – OFF = laite sammutetaan, ON = käynnistä laite 
* Group Call: No/Yes – No = ei ryhmäkutsu, Yes = komento menee kaikille saman house coden laitteille kerralla.
* Dim Value: 0 – himmennysarvo
